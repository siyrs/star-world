# 星的世界 · Star World

《星的世界》是一款使用 Godot 4 和 GDScript 开发的 Windows 第一人称 3D 像素沙盒游戏。玩家可以选择五种世界，探索随机地形，采集、合成、建造并在昼夜循环中生存。世界的方块修改、玩家位置、背包和生存状态都会持久保存。

## 已实现玩法

- 30 种体素方块，包括地形、水/岩浆、矿物、玻璃、台阶、楼梯与建筑材料。
- 16×16 水平分块、确定性 Seed 生成、动态加载/卸载与稀疏修改存档。
- 五张地图：星辰大陆、荒漠遗迹、极寒冰原、天空群岛、深渊世界。
- 第一人称移动、跳跃、重力和碰撞；左键破坏、右键放置。
- 武器攻击、动物/僵尸受伤与死亡掉落；选中食物后右键食用。
- 36 格背包、9 格快捷栏、62 种可注册物品，支持堆叠、交换、收集和序列化。
- 随身合成、工作台和熔炉三种工位，42 个会真实消耗输入并产生输出的配方。
- 生命、饥饿、饱和、自然恢复、饥饿伤害、死亡和重生。
- 10 分钟默认昼夜循环，太阳、环境光和天空颜色随时间变化。
- 鸡、牛、猪和僵尸：程序化体素模型、漫游/追击 AI、血量、伤害、死亡动画和掉落物。
- 主菜单、五地图选择、多存档浏览、设置、HUD、背包、合成、暂停与死亡 UI。
- 无外部版权资产：生物模型和环境/方块/生物音效均在运行时程序化生成。

## 快速开始

1. 安装 Godot 4.3 或更高版本（本项目已用 Godot 4.7 验证）。
2. 在 Godot Project Manager 中导入根目录下的 `project.godot`。
3. 按 `F6/F5` 运行，或直接启动已导出的 `StarWorld.exe`。
4. 在主菜单选择“开始游戏”，选地图、输入世界名与 Seed，然后创建世界。

详细命令行构建和 Windows 导出方法见 [BUILD.md](BUILD.md)，模块与存档合同见 [ARCHITECTURE.md](ARCHITECTURE.md)。

## 世界存档

存档使用带 `save_version` 的 JSON，每个世界一个目录。Windows 默认位置是：

```text
%APPDATA%\Godot\app_userdata\星的世界\worlds\<world-id>\world.json
```

存档包含地图类型和 Seed、稀疏方块修改、玩家位置、背包、生命/饥饿和昼夜时间。文件先写入临时路径再原子替换，降低异常退出造成损坏的风险。

## v1.0 边界

本版是完整可玩的简化单人核心版本。水和岩浆为静态体素；生物 AI 使用有限状态机而非全地图导航；台阶/楼梯以简化方块网格呈现。多人、红石类自动化和商业级大型内容库不在 v1.0 范围内。
